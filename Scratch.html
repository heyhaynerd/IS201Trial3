Star Color Facts

Professional Scratch







  • Red Stars
      Temperature: 2,000 C - 3,600 C
      Examples: Antares, Betelgeuse

  • Orange Stars
      Temperature: 3,600 C - 5,000 C
      Examples: Aldebaran, Arcturus

  • Yellow Stars
      Temperature: 5,000 C - 6,000 C
      Examples: Proxima, the Sun

  • White Stars
      Temperature: 7,500 C - 11,000 C
      Examples: Sirius, Vega

  • Blue Starts
      Temperature: 11,000 C - 28,000 C
      Examples: Rigel, Spica, Bellatrix
